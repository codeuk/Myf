# Myf
Myf is a RAT &amp; Information Logger created through a Telegram Bot

# Usage

	🕸️ Discord
		/tokens       [IN PROGRESS]
		/discordinfo  [IN PROGRESS]

	📚 Files
		/execute <file>
		/download <file>
		/remove <file>

	🌐 Browsers
		/history
		/passwords

	📸 Spying
		/screenshot

	📁 Directories
		/dir  
		/cwd
		/cd <dir>

	👾 Tasks
		/tasklist
		/endtask <pid>

	💥 Misc
		/cmd <command>
		/startup
